# 🎵 VibeStream

O **VibeStream** é uma plataforma de streaming de música inovadora que une a paixão pela música com a interatividade social. Além das funcionalidades clássicas de um player, o projeto foca em gamificação e descoberta inteligente de sons.

---

## ✨ Funcionalidades Diferenciais

### 🔍 Identificação de Áudio Exterior
Esqueça ter que abrir outros aplicativos para saber que música está tocando. O VibeStream utiliza a captura de áudio do microfone para identificar faixas instantaneamente dentro da própria plataforma.

### 🏆 Arena de Fãs (Competição entre Amigos)
Crie grupos ou entre em competições com seus amigos para ver quem é o maior entusiasta! 
- **Tempo de Escuta:** Ranking baseado em minutos ouvidos em tempo real.
- **Duelo de Fãs:** Sistema que identifica quem mais consome a discografia de um artista específico.
- **Badges de Conquista:** Ganhe medalhas por marcos alcançados e tempo de fidelidade.

## 🚀 Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/vibestream.git
    ```
2.  **Acesse a pasta do projeto:**
    ```bash
    cd vibestream
    ```
3.  **Execução:**
    Como o projeto utiliza tecnologias Front-End padrão, basta abrir o arquivo `index.html` em qualquer navegador moderno.

> [!IMPORTANT]
> A funcionalidade de **Identificação de Áudio** requer que o utilizador conceda permissão de acesso ao microfone no navegador.
